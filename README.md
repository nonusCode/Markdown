# Markdown

> 텍스트 기반의 마크업언어



### Header

> 문서의 제목,  부제목, 글머리
>
> `#`의 개수,  `<h1~6></h1~6>`의 형태로 표현 가능
>
> `Ctrl + 1~6`의 단축키 사용 가능

```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```



### BlockQuote

> 인용문
>
> `>` 입력 후 `Space` 입력
>
> 다른 마크다운 요소 포함 가능



### List

> 목록

* 순서가 있는 목록

  > `숫자.` 입력 후 `Space` 입력
  >
  > `Shift + Tab`, `Tab`키를 눌러 상위 항목이나 하위 항목으로 이동 가능

* 순서가 없는 목록

  > `*`,  `+`,  `-` 입력 후 `Space` 입력
  >
  > `Shift + Tab`, `Tab`키를 눌러 상위 항목이나 하위 항목으로 이동 가능
